HW06 -- "Hello, World!" in Scheme, NetLogo, and Python
2021-09-28

Scheme: "Hello, World!"

NetLogo: write "Hello, World!"

Python: print ("Hello, World!")

Similarities: 
- The message has to be in quotes.

Differences:
- Scheme doesn't require a keyword like write (for NetLogo) or print (for Python)
- Python requires parentheses around the message.
